title: FSC Replacer
project_url: https://github.com/KonradJam/FSC_replacer
categories:
  - automatisation
  - graphic
  - dtp
technologies:
  - Python
  - PySide6
  - OpenCV
  - PyMuPDF
problem: Manually replacing the FSC mark in a large number of PDF files is
  time-consuming and error-prone.
solution: A desktop application automatically detects a target graphic in PDF
  files and overlays a new asset in the same position in batch mode.
impact: It reduces manual work in repetitive update tasks and can also be used
  for other logos or recurring graphic elements.
